## Getting started

Install with npm

#### Prerequisites

NPM

```bash
npm install npm@latest -g
```

#### Installation

1. Clone the repository

```bash
  git clone https://github.com/subinho/portfolio-2024.git
```

2. Install npm packages

```bash
  npm install
```

3. Start server

```bash
npm run dev
```
